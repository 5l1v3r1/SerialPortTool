# SerialPortTool
A  Cross-platform serial test tool.

## Dependencies
Required:
* [cmake](http://www.cmake.org) - buildsystem

## Build on Linux
~~~
git clone https://github.com/Joker2770/SerialPortTool.git
git submodule update --init --recursive
cd SerialPortTool
mkdir build
cd build
cmake ..
make
./SerialPortTool
~~~

## Build on windows
~~~
git clone https://github.com/Joker2770/SerialPortTool.git
git submodule update --init --recursive
cd SerialPortTool
~~~
build with cmake-gui.
